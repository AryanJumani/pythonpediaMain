Data Types and Variables

There are 4 basic data types in python:

  • Text
  • Numeric
  • Sequences
  • Boolean
For now, we will just be focusing on the first two.

Strings

A string is a text data type which is represented as a sentence, a word, or any alphanumberic letter.
Your first code on printing Hello World itself included a string which had to be printed.

To put it simply, anything between double quotations("") or single quotations('') is a string

Numbers

A number is any numeric value, whether it is an integer or a floating point value(a decimal number) or a complex number

Now let us learn what are variables.
Variables is certain data that the computer stores. For example, John has five apples, so we can create a variable called noOfApples and print it with the following syntax:

Example

noOfApples = 5
print(noOfApples)
Try it out!

Another variable, but now it holds a string

Example

s = "This is a string"
print(s)
Try it out!

To know the data type of a variable, use the type function

Example

noOfApples = 5
print(type(noOfApples))
Try it out!
