Booleans

Unlike other data types which can hold several different values, boolean can store only two values: True and False.

When you compare two values, a Boolean value will be returned to the user.

Example

print(5>9)
print(5>4)
print(5==3)
Try it out!

Conditions

Python supports the following conditions which return a Boolean value:

a==b A is equal to B
a!=b A is not equal to B
a>b A is greater than B
a<b A is lesser than B
a>=b A is greater than or equal to B
a<=b A is lesser than or equal to B

In the if statement, Python will return a Boolean value. Learn more about if statements here

Example

a = 10
b = 5
if(a>=b):
    print("A is greater than or equal to B") #It returned True
else:
    print("A is not greater than or equal to B") #It returned False
Try it out!

Bool function

The bool() function checks the value entered within the brackets and returns either True or False.

Example

print(bool(100))
print(bool("Hello World"))
Try it out!

Most of the times, the printed value will be True. However, sometimes the values will be False when the value is empty, such as: (), [], {}, "", 0, the value None and False itself.

Example

print(bool(()))
print(bool([]))
print(bool({}))
print(bool(""))
print(bool(0))
print(bool(None))
print(bool(False))
Try it out!
