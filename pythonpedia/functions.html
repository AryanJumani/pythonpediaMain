Functions

Functions are lines of code which only run when they are called.

The function can return data which can then be printed.
It is a way to avoid repetition of code.

Creating Functions

Functions are created by the use of the def keyword.

Example

def aFunction():
    print("This is running from a function!")

Calling Functions

To call a function in Python, you simply type the name of the function whenever you want to run that piece of code.

Example

def aFunction():
    print("This is running from a function!")
aFunction()
Try it out!

Return values

As stated earlier, functions return values. This is done using the return statement.
This is useful when you just want to access a value and not print it.

Example

def aFunction():
    return "This is being returned from the function. So, it won't print unless the function is printed."
aFunction()
print(aFunction())
Try it out!

Arguments

Sometimes, the function needs some information to work as intended. This is done using arguments.

They are written within the brackets of the function, after the function name.
You have to separate arguments with a comma if there are more than one.

Example

def aFunction(theinput):
    x = theinput * 5
    return x
a, b, c = map(int, input("Enter three numbers separated by a space: ").split())
print(aFunction(a))
print(aFunction(b))
print(aFunction(c))
Try it out!

Note

Arguments are also called parameters.

If you have asked for 2 arguments in the function, then Python will need two arguments itself when the function is called.
Any more or any less will result in an error.

Example

def aFunction(a, b):
    x = a * b
    return x
a, b = map(int, input("Enter two numbers separated by a space: ").split())
print(aFunction(a, b))
Try it out!

This did not give any error. However, the code below will give an error.

Example

def aFunction(a, b):
    x = a * b
    return x
a, b = map(int, input("Enter two numbers separated by a space: ").split())
print(aFunction(a, b, 5))
Try it out!

If you do not know how many arguments will be passed in your function, use a * before the argument name in the function.
This will allow you to enter multiple arguments which will be stored in a tuple of the argument name.

Example

def aFunction(*a):
    x = 1
    for i in range(len(a)):
        x = x*a[i]
    return x 
a, b = map(int, input("Enter two numbers separated by a space: ").split())
print(aFunction(a, b, 5))
Try it out!

Note

The value of x in the above example is set to 1 and not 0, since 0 multiplied by any number would return 0 itself.

You can also input arguments by telling what argument name refers to what value. This is called as keyword arguments.

Example

def aFunction(a, b, c):
    x = a + b * c
    return x
no1, no2, no3 = map(int, input("Enter three numbers separated by a space: ").split())
print(aFunction(a = no1, c = no2, b = no3))
Try it out!

If you do not know how many keyword arguments will be input by the user, use ** before the variable name in the function.
This will save the entire input as a dictionary.

Example

def aFunction(**thisdict):
    x = 1
    for i in thisdict:
        x = x*thisdict[i]
    return x
no1, no2, no3 = map(int, input("Enter three numbers separated by a space: ").split())
print(aFunction(a = no1, b = no3, c = no2))
Try it out!

Default parameter value

If you pass a function without an argument, it can use the default parameter value if you assign it.

Example

x = 1
def aFunction(a=2):
    global x
    x = x * a
    return x
no1, no2, no3 = map(int, input("Enter three numbers separated by a space").split())
print(aFunction(no1)) #Prints 1*no1
print(aFunction()) #Prints no1*2(default value)
print(aFunction(no2)) #Prints no1*2*no2
print(aFunction(no3)) #Prints no1*no2*no3*2
Try it out!

Note

The keyword global just makes use of the variable x which is outside the function and has not been passed as an argument.
We will learn more about local and global variables later.
