For Loop

The other loop in python is the for loop

The for loop is normally used to go through or iterate over a list, tuple, dictionary, set, or string

Example

l = ["German", "English", "French"]
for i in l:
    print(i)
Try it out!

Note

In the for loop you do not need to create an integer which is being incremented.
Also note, the break statement works the same way as it does in the for loop.

Continue Statement

The continue statment stops the current iteration and moves onto the next one.

Example

l = ["German", "English", "French"]
for i in l:
    if(i == "English"):
        continue
        #English will be skipped
    print(i)
Try it out!

Range

The range statement used in the for loop to iterate an integer from the starting to ending values provided by the user itself.
It is similar to the while loop when using an integer to iterate through it, however in the for loop, you do not need to increase the variable's value.

Example

l = ["German", "English", "French"]
for i in range(len(l)):
    print("List index: " + str(i) + "\nList item at index: " + l[i] + "\n\n")
Try it out!

Note

If not given two values within the range brackets, Python assumes the starting value to be 0 and the entered value to be the ending value.

You can specify the starting value by using the following syntax

Example

l = ["German", "English", "French"]
for i in range(1, len(l)):
    print("List index: " + str(i) + "\nList item at index: " + l[i] + "\n\n")
Try it out!

You can also set the step value in the range brackets by a comma. The step value is just like the step value in string and list slicing.

Example

l = ["German", "English", "French", "Russian", "Mandarin", "Hindi", "Spanish"]
for i in range(1, len(l), 2):
    print("List index: " + str(i) + "\nList item at index: " + l[i] + "\n\n")
Try it out!

Else statements

The for loop's else statement is run when the for loop finishes

Example

l = ["German", "English", "French"]
for i in range(len(l)):
    print("List index: " + str(i) + "\nList item at index: " + l[i] + "\n\n")
else:
    print("The For Loop has finished running.")
Try it out!

Note

The else statement will not run if the for loop is ended by a break statement.
