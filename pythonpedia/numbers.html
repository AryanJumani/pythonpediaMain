Numbers in python

There are three numeric types of data in Python:

  • int
  • float
  • complex

Example

a = 2 #int
b = 2.5 #float
c = 2j #complex

Int

An int(integer) type is any number, positive or negative, without any decimals or fractions.

Float

A float(floating point number) is a number, positive or negative, with decimals.
Floats can also be scientific numbers with an 'e' to indicate the power of 10.

Complex

Complex numbers are written with 'j' as their imaginary part instead of 'i'.

Type conversion

You can convert from one type to another using the int(), float(), and complex() methods

Example

a = 2 #int
b = 2.5 #float
c = 2j #complex

#int to float
d = float(a)
#int to complex
e = complex(a)
#float to int
f = int(b)
#float to complex
g = complex(b)
Try it out!

Note

You cannot convert from complex to any other numeric type

Random Numbers

To make random numbers within a range, python has a random module which needs to be imported with the import system

Example

import random
#print a random number between 1 and 100
print(random.randrange(1, 100))
Try it out!
