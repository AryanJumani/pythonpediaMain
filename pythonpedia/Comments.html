Comments

Comments are used in python to:

  • Explain code
  • Make the code more readable
  • Check to see if code is working by preventing its execution

Comments are ignored by Python and so it can be used only be the developers to know what a block of code does. It can also be helpful to debug code.

In python, the single-line comments start with a #

Multi-line comments start and end with three double quotes """ or three single quotes '''

Example

#This is a comment

"""This is a multi-line comment
Which can last for several lines until it is closed
By three more quotes"""

'''They can be double quotes
Or single quotes too'''
