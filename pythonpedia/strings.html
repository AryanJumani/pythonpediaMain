Strings

Strings, as said earlier, is any alphanumeric letter.
In python, it is represented within 2 double quotations(" and ") or 2 single quotations(' and ')
"string" is the same as 'string'.

Example

print("string")
print('string')
print(type("string"))
print(type('string'))
Try it out!

Multiline strings

Strings can extend to several lines. However, you cannot use 2 double or single quotations to define it.
To tell the computer that there is a multiline string, use 3 double or single quotations.

Example

s = """This is a
multi-line string."""
print(s)
Try it out!

String Slicing

Strings are arrays. To access a paarticular character in the string, we use square brackets.
This is known as String Slicing.

Example

s = "Hello World"
print(s[1])
Try it out!

Note

To learn more about String Slicing, refer to List Indexing.
They are similar except for the fact that list items are separated by commas.

Upper Case

To get the entire string into upper case, use the upper() method.

Example

s = "Hello World"
print(s.upper())
Try it out!

Lower Case

To get the entire string into lower case, use the lower() method.

Example

s = "Hello World"
print(s.lower())
Try it out!

Remove Whitespaces

To remove whitespaces( ) from the beginning or end of the string, use the strip() method.

Example

s = " Hello World  ! "
print(s.strip()) #Returns "Hello World  !"
Try it out!

Replace strings

You can replace a string with another string using the replace() method.
It takes two arguments, the first is the string which needs to be replaced and the second is the new string.

Example

s = "Hello World!"
print(s.replace("Hello", "Bye"))
Try it out!

String split

To separate a string based on a certain character, use the split() method.
It will return it as a list.

Example

s = "Hello, World!"
print(s.split(",")) #Returns ['Hello', 'World!']
Try it out!

String concatenation

You can combine two different strings using the + operator.

Example

a = "Hello"
b = "World!"
c = "This is string concatenation"
print(a + " " + b + " " + c + "!")
Try it out!

Join method

To add a certain string between any iterable(list, tuple, dictionary keys) and convert the iterable into a string, use the join() method.

Example

 theList = ["Join", "this", "list", "with", "a", "dash"]
s = "-".join(theList)
print(s)
Try it out!
