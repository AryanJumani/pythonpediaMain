Tuples

Just like lists, tuples store multiple items in a single variable

The only difference between a tuple and a list is that lists can be changed while tuples are unchangeable.

There is also a slight syntax difference while creating tuples.

They start and end with round brackets instead of square brackets

Example

thetuple = ("Apple", "Banana", False, 10)
print(thetuple)
Try it out!

They can be accessed the same way as a list since all items are ordered. That is, each item has an index and can be accessed using it.

Tuple to List and back

Since tuples cannot be changed(immutable), the only way to do so is to convert the tuple into a list, change the list and then convert it back into a tuple.

Example

thetuple = ("Apple", "Banana", False, 10)
thelist = list(thetuple)
thelist[-1] = 50
thetuple = tuple(thelist)
print(thetuple)
Try it out!

Length of a tuple

Exactly like a list, the length of a tuple can be known the same way, that is, using the len() function

Example

thetuple = ("Apple", "Banana", False, 10)
print(len(thetuple))
Try it out!

Count method

You can count the number of occurences of an item in the list.

Example

thetuple = ("Apple", "Banana", False, 10)
print(thetuple.count(False))
Try it out!

Index Method

You can come to know the index at which an item is at, using the index method

Example

thetuple = ("Apple", "Banana", False, 10)
print(thetuple.index(False))
Try it out!

Note

It only gives the index of the first occurence of the value needed.

Unpacking A Tuple

While creating a tuple, we assign values to one integer. This is known as "packin" a tuple.
But, in python we can also "unpack" a tuple. Which is, assigning variable to each of the items in the tuple.

Example

thetuple = ("Apple", "Banana", False, 10)
(fruit1, fruit2, boolean, number) = thetuple
print(fruit1)
print(fruit2)
print(boolean)
print(number)
Try it out!
