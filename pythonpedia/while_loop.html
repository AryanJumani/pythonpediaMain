Python Loops

Python has mainly two loops:

  • while loop
  • for loop

While Loop

The while loop will keep on running as long as a condition provided is true

It is also used in games as the main loop which never stops

Example

i = 1
while(i <= 10):
    print(i)
    i += 1
Try it out!

Note

Do not forget to increase i by 1 otherwise the loop will run forever

Break statement

The break statement ends the loop even if the condition is true

Example

i = 1
while(i <= 10):
    print(i)
    if(i == 6):
        break
    i += 1
Try it out!
