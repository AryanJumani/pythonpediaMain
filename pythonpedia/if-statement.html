The If Statement is used to ask the computer a question, called as the condition.

The computer can only answer the question as a yes(true) or no(false)

If it is true, then the computer will do the following code which is written with an indentation.

If and Else statement

The condition must return a boolean value otherwise it will return an error.

If the If statement returns false, it will run the Else statement automatically unless there is an elif statement

The syntax for the if and else statement is as follows:

Example

if(5>2):
    print("This is true. 5 is greater than 2")
Try it out!

Elif statement

The Elif statement is short for else if which means that if the if statement reutrns false, it will move onto the elif condition instead of the else statement. There can be many elif statements.

This demonstration below will show you the syntax for elif and give a better understanding of the if statement.

Example

a = int(input("Enter a number")
b = int(input("Enter another number")
Try it out!

Logical Operators

Sometimes, there may be more than one condition which you might have to put, for that there are logical operators.

And

The And operator is a logical operator which works like the AND logic gate. To put it simply, with the usage of and in the condition, it will return true only if both the conditions provided return true.

Example

a = int(input("Enter a number")
b = int(input("Enter another number")
c = 10
Try it out!

Or

The Or operator is a logical operator which works like the OR logic gate. To put it simply, with the usage of or in the condition, it will return true when any one of the conditions returns true.

Example

a = int(input("Enter a number")
b = int(input("Enter another number")
c = 10
Try it out!

Nested If

You can even have if statements inside if statements. This is known as nested if statements

Example

a = int(input("Enter a number")
if(x>5):
    print("X is greater than 5")
    if(x>10):
        print("X is also greater than 10")
    else:
        print("However, it is not greater than 10") 
Try it out!

Pass statement

If statements' content cannot be left blank, unless you pass a parameter called pass

Example

if(5>2):
    pass
else:
    print("Something went wrong with mathematics")
Try it out!
