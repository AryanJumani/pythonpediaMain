Classes and Objects

Classes are the backbone of python.
Everything in python is based on classes and objects.

Classes are pieces of code that define objects.
Classes are mainly used to contain several functions and to classify those functions.

An example of a class is a car. A car needs to perform several different functions to operate properly: to move, to regualte temperature, to keep check of speed, to keep check of fuel/battery, etc. All of this combined into one is a car.

Creating Classes

Here's how to create a class. Note: this class does not do anything since classes have to be called just like functions before it does anything.

Example

class thisClass():
    x = 5 

Creating objects

To use classes, objects need to be created. Objects are how classes are called.

Example

class thisClass():
    x = 5
obj = thisClass()
print(obj.x)
Try it out!
