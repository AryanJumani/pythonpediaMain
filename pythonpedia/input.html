Input statement

To ask the user something, python takes an input from the user using the input method

We are going to use the string to integer conversion we learned earlier.

Example

a = int(input("Enter a number: "))
print(a)
Try it out!

Multiple inputs on a single line

If you need to take multiple inputs in python on a single line, the map function and the split method allows you to do so

Since the split method's default parameter is a space, the inputs will have to be separated by a space. You can also enter whatever parameter you want.
