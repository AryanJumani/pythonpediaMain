First we need to download Python.

Note

If you already have python installed, proceed onto the next part.
To check if python is installed on your computer,

Windows:

Type in the search bar: cmd.exe and press enter. Then type
python --version

Mac:

Go to Application > Utilities > Terminal and press enter. Then type
python --version

Linux:

Open terminal. Then type
python --version

Click this link to go to the official site and download the latest version.
Click on Customize Installation and make sure everything is ticked.
Click Next and Then Install. You could change the location of your python installation using the Browse button
Wait for Python to install.

Note

If you do not want to download python it is fine. You can use any online IDE such as trinket, Programiz or onlinegdb.
