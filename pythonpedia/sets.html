Sets

A set is the third data type used to store multiple items. So far we have covered Lists and Tuples.

A set is unordered and unindexed

Unlike lists, duplicate items are not allowed in sets

Like lists, their items can be changed.

They are written with curly brackets

Example

theset = {"Apple", "Banana", False, 10, False}
print(theset)
Try it out!

Note

Remember, sets are unordered, so the order of the output might be different to what was given as input

Access items in a set

Since the items in a set are unindexed, you cannot use the square brackets to access the items in the set.

So to access it, you have to use a for loop.

Example

theset = {"Apple", "Banana", False, 10, False}
for i in theset:
    print(i)
Try it out!

Add items to a set

Sets can be changed. To add items to a set, simply use the add() method.

Example

theset = {"Apple", "Banana", False, 10, False}
theset.add(50)
print(theset)
Try it out!

Add another sequence to a set

Use the update() method to add any sequence(set, tuple, dictionary, list) to a set

Example

theset = {"Apple", "Banana", False, 10, False}
thetuple = ("Cucumber", "Dragonfruit")
theset.update(thetuple)
print(theset)
Try it out!

Remove items from a set

To remove an item from a set, simply use the remove() method.

Example

theset = {"Apple", "Banana", False, 10, False}
theset.remove(10)
print(theset)
Try it out!

You can also use the pop() method to remove the last item in the set.
However, since sets are unordered, you will not know which item has been removed.

Example

theset = {"Apple", "Banana", False, 10, False}
theset.pop()
print(theset)
Try it out!
