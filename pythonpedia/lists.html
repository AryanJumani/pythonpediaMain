Lists

Lists store several items in a single variable.

Lists use square brackets - []

They can store several data types.

They can even be duplicate values

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l)
Try it out!

Length of A List

You can find out the number of items in a list using the len() function

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(len(l))
Try it out!

List Constructor

Instead of using the square brackets, you can also use the list() constructor to create a new list

Example

l = list(("English", "Mathematics", 500, True, "Economics", True)) #Note the double Round brackets
print(l[4], l[-2])
Try it out!

Count method

you can count the number of occurences of an item in the list.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l.count(True))
Try it out!

Indexing

List items are numbered. For example, the first item in the list has been given a number(index) and so on.

The first item is given the index [0]. The second item is [1] etc.

You access the list item using the following syntax:

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l[0], l[3])
Try it out!

Negative indexing

Python also has negative indexes - [-1], [-2] etc.

[-1] refers to the last item in the list. [-2], the second last item and so on.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l[4], l[-2])
Try it out!

Range of Indexes

You can print several items from the list by asking to print a range of indexes(start to end)

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l[1:4])
Try it out!

Note

While using a range of indexes, the starting index([1] in this case) is included, but the ending index is excluded.

You can not include the starting index value to start the range from the first item in the list.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l[:4])
Try it out!

By not including the ending index value, the range ends at the last item in the list.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l[2:])
Try it out!

You can also skip an index(for example, to print only the 1st, 3rd, 5th etc. items in the list) by stepping the list by 2

To do that, the syntax is:

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l[::2])
Try it out!

Index Method

You can come to know the index at which an item is at, using the index method

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
print(l.index(True))
Try it out!

Note

It only gives the index of the first occurence of the value needed.

Change List Items

It is possible to change specific list items using their index value.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l[1] = "Physics"
print(l)
Try it out!

Change a range of indexes

You can also change a specific range of list items.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l[1:4] = ["Physics", 200, False]
print(l)
Try it out!

Remember, the ending index value is excluded


If you enter more items than you intended to replace, then the items will be inserted without any replacement of the other items and the other items will also move.
This also means that the lenght of the list will change.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l[1:2] = ["Physics", 200, False]
print(l)
Try it out!

If you enter less items than you intended to replace, then the other items included in the range, but not replaced will be removed.
This will change the length of the list.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l[1:5] = ["Physics"]
print(l)
Try it out!

Append Method

You can add elements to the end of your list by using the append method

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l.append("Physics")
print(l)
Try it out!

Insert method

You can add elements to your desired index value using the insert method
This would change the length of a string

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l.insert(1, "Physics")
print(l)
Try it out!

Extend method

You can use the extend method to add any iterable object including lists, tuples, dictionaries, sets etc. The last three will be covered later.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
fruits = ["Apple", "Banana"]
l.extend(fruits)
print(l)
Try it out!

Remove method

You can remove whichever item from the list you want using the remove method

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l.remove(True)
print(l)
Try it out!

Note

It only removes the first occurence of the item that needs to be removed

Pop method

The pop method removes an item from the list at the specified index.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l.pop(3)
print(l)
Try it out!

Note

If no index value is specified in the pop method, it will remove the last item in the list by default.

The del keyword can also delete items at a certain index or the list itself

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
thelist = ["Apple", "Mango", 10]
del(l[1])
del(thelist) #Will give an error while trying to print because it has been deleted
print(l)
print(thelist)
Try it out!

Clear method

Using the clear method, the entire list can be emptied. That is, the list is not deleted, but it is just has no items in it.

Example

l = ["English", "Mathematics", 500, True, "Economics", True]
l.clear()
print(l)
Try it out!

Sort method

The sort method can sort the list by ascending order by default. It works for numbers and strings

Example

l = ["English", "Mathematics", "Economics", "Physics"]
l.sort()
print(l)
Try it out!

You can also sort the list by descending order using the reverse keywork

Example

l = ["English", "Mathematics", "Economics", "Physics"]
l.sort(reverse = True)
print(l)
Try it out!
